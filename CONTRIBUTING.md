# Contributing to LOs Dataset Creator

Thank you for your interest in contributing to this project! We welcome contributions from the community to help improve the quality and scope of educational dataset generation.

## How to Contribute

### 1. Reporting Bugs
- Use the GitHub Issues tab to report bugs.
- Provide a clear description of the issue and steps to reproduce it.

### 2. Suggesting Enhancements
- If you have ideas for new features (e.g., more topic categories, support for other LLMs), please open an issue to discuss them.

### 3. Pull Requests
- Fork the repository.
- Create a new branch for your feature or bug fix.
- Ensure your code follows the existing style and includes comments.
- Update the README if necessary.
- Submit a pull request with a clear description of your changes.

## Development Setup
1. Install development dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Test your changes locally before submitting.

## Code of Conduct
Please be respectful and professional in all interactions within this project.
